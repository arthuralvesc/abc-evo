# ABC EVO

Este projeto foi construído como parte do Desafio Técnico da vaga de Desenvolvedor Frontend Jr.

A versão do Angular utilizada foi a 17, e os componentes são standalone.

O formulário de cadastro foi feito usando Reactive Forms.

Devido a complexidade da tarefa ser relativamente baixa, não foi utilizada nenhuma service para manipulação dos dados. Também não foi utilizada nenhuma biblioteca de programação reativa.

Foram desenvolvidos testes unitários para as principais funções da aplicação (persistência de dados no localStorage, validação dos campos do formulário, utilização correta dos dados na tela de informações).
